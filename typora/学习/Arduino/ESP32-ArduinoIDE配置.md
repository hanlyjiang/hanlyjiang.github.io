# ESP32-Arduino IDE配置

## 参考文档

* [Installing — Arduino-ESP32 2.0.0 documentation (espressif.com)](https://docs.espressif.com/projects/arduino-esp32/en/latest/installing.html)



## 安装配置

1. 添加仓库地址

```
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
```

3. 打开 Board Manager， 搜索ESP32 并安装
