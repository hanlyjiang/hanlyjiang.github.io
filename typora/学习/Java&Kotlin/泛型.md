## 问题- T 和 `<T> ` 的区别

```java
T create(String name, Action<? super T> configureAction) throws InvalidUserDataException;

<T> void configure(String name, Action<? super T> action);
```



`<T> void configure(String name, Action<? super T> action);`中，`Action<? super T> action` 对应于kotlin中的`Action<in T>`
